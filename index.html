What is this?

This is the source code for the Hackasoton landing / presetation page template.

How to use it?

If you wanna see it locally, just do : 

$ git clone http://github.com/sabinmarcu/hackasoton-wow.git
$ cd hackasoton-wow
$ python -m SimpleHTTPServer

Now go to [http://localhost:3000/](http://localhost:3000/) and see it for yourself.

If you wanna tinker with the code, just do : 

$ git clone http://github.com/sabinmarcu/hackasoton-wow.git
$ cd hackasoton-wow
$ git checkout -b master
$ git pull origin master

From now on, you can simply do :

$ npm install -g grunt-cli
$ sudo grunt

If port 80 is available, it will use that. If port 80 is not available, or you don't have root permission (unlikely), just do $ grunt superalternate and it will use port 8000. This way (both of the options) the nodejs server bundled will compile the app / interface every time you make a request. All of the necessary stuff is located in the src/ folder. Data files like languages, styles, images, fonts, views, whatever are located in the src/data folder, while the javascript (livescript, coffeescript, whateverscript) logic files are located in the src/classes folder. Have fun! :)
